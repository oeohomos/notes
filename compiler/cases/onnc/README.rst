========================================
ONNC (Open Neural Network Compiler)
========================================


.. contents:: 目錄


介紹
========================================

:網站: https://onnc.ai/


ONNC 是臺灣 Skymizer 做的編譯器，
可以把 ONNX 格式的 deep learning model 針對特定硬體做優化，
硬體商只要給出該硬體每個操作的 cost model，
編譯器就可以把 deep learning model 優化到該平台上。
